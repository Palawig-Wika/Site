Gabay

Ang Gabay (Guide) ay naglalaman ng mga gabay sa paggamit ng Palawig-Wika.

Paano Gumamit ng Educational Website para sa Lesson (Para sa mga Mag-aaral)

  1. Buksan ang Internet Browser

    Una, magbukas ng browser sa iyong computer o cellphone tulad ng:

    • Google Chrome
    • Mozilla Firefox
    • Microsoft Edge

    Sa address bar, i-type ang website ng learning site. Halimbawa:

    • Palawig-Wika

    Pagkatapos ay pindutin ang Enter.

  2. Pumunta sa Panimulang Pahina

    Pagbukas ng Palawig-Wika, makikita ang Panimulang Pahina. Karaniwang makikita dito ang:

    • Pamagat na Palawig-Wika
    • Ibig sabihin ng Palawig-Wika at History na galing kay Dr. Jose Rizal
    • Mga Litrato
  3. Pumili ng Aralin

    Hanapin ang Aralin na gusto mong pag-aralan. Halimbawa:

    • Korido
    • Aralin

    Pindutin ang mga kahon upang lumabas ang pahinang nais mong pag-aralan.

  4. Maghanap ng Aralin

    Sa loob ng kahon ay makikita ang mga aralin. Nakasaad ang buong akda ng Ibong Adarna.

    Halimbawa:

    • Aralin 1–15
    • Aralin 16–27
    • Aralin 28–37
    • Aralin 38–46
  5. Basahin ang mga Aralin

    Kapag napili mo na ang lesson, maaari mong:

    • Basahin ang explanation
    • Tingnan ang examples at images

    Basahin nang mabuti ang bawat bahagi para mas maintindihan ang mga aralin.

    Sagutan ito para masubukan ang iyong natutuhan.

    Maaaring sukatin ang lawak ng inyong kaalaman gamit ang mga matatalinghagang salita.

Gamitin ang menu sa taas para mag-navigate sa iba't ibang aralin.
