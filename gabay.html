Gabay

Ang Gabay (Guide) ay tutulong sa iyo kung paano gamitin ang Palawig-Wika nang madali at maayos.

📘 Paano Gamitin ang Website (Para sa mga Mag-aaral)

  1. 🌐 Buksan ang Internet Browser

    Magbukas ng browser sa iyong cellphone o computer tulad ng:

    • Google Chrome
    • Mozilla Firefox
    • Microsoft Edge

    Sa address bar, i-type ang Palawig-Wika at pindutin ang Enter.

  2. 🏠 Pumunta sa Panimulang Pahina

    Makikita mo rito ang mga sumusunod:

    • Pamagat na Palawig-Wika
    • Maikling paliwanag at kasaysayan
    • Mga larawan
  3. 📚 Pumili ng Aralin

    Piliin ang aralin na gusto mong pag-aralan.

    Halimbawa:

    • Korido
    • Iba’t ibang Aralin

    I-click ang kahon upang buksan ang aralin.

  4. 🔍 Hanapin ang Iyong Aralin

    Sa loob ng kahon, makikita ang mga bahagi ng aralin mula sa Ibong Adarna tulad ng:

    • Aralin 1–15
    • Aralin 16–27
    • Aralin 28–37
    • Aralin 38–46
  5. 📖 Basahin at Sagutan ang Aralin
    • Basahin ang paliwanag
    • Tingnan ang mga halimbawa at larawan
    • Sagutan ang mga gawain

    💡 Tip: Basahin nang mabuti para mas maintindihan ang aralin.

    📝 Sagutan ang mga tanong para masubukan ang iyong natutuhan.

    📊 Subukan ang iyong kaalaman gamit ang mga matatalinghagang salita.

    📌 PAALALA: Kung nais mong bumalik sa Unang Pahina, i-click ang "Palawig-Wika" na makikita sa itaas ng pahina.

🧭 Gamitin ang menu sa itaas para lumipat sa iba't ibang aralin.
